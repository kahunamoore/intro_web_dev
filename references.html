  • Package Management
    Brew (macOS)
    Installs and updates commonly used tools for software development, system utilities, system services. The Brew command itself and the packages it installs are primarily command-line programs. A lot of advanced tools do not have GUIs and operate better on the command line because the output of one can be piped into the input of another allowing very powerful chaining of operations.
    Website
    PowerShell (Windows)
    Windows only command line oriented for software development and system/network maintenance.
    Some commands require Administrator access so launch with Admin privledges (right-click app, choose Run as... Administrator). Should be used with caution.
    Has a powerful shell scripting language in the Algol/C tradition.
    iTerm 2 (macOS)
    Very nice (and free) alternative to the macOS default Terminal app.
